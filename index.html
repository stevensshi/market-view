How it's built

firehose news mail + research newsletters + web search + viral social ↓ EXTRACT per-record structuring (budget LLM tier) ↓ CLUSTER + SYNTHESIZE per-sector fan-out, market-wide ROLLUP (frontier LLM) ↓ VERIFY → RANK every claim needs a source; materiality = weighted signals ↓ RENDER one self-contained HTML file — the page you're about to open
  • Event families (事件族): theses driven by one underlying event are grouped, never merged — opposite-direction logics stay tradable.
  • Consensus ↔ contrarian: each thesis carries for/against evidence bars, so the minority view is visible instead of averaged away.
  • Nightly autonomy: a scheduled headless agent refreshes sources, rebuilds, passes a privacy gate, and publishes — no human in the loop.
